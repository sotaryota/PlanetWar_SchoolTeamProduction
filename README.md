## 操作方法
### ※コントローラーの接続が必要です
- 移動・メニュー選択

  左スティック
- カメラの回転
  
  右スティック
- ロックオン

  RBボタン・R1ボタン
- 敵に注目

  LBボタン・L1ボタン
- 惑星を掴む・決定

  Aボタン・×ボタン
- 惑星を投げる ※惑星所有時
  
  Bボタン・〇ボタン  
- 惑星を離す ※惑星所有時

  Yボタン・△ボタン
- 回避 ※惑星非所有時

  Xボタン・□ボタン
