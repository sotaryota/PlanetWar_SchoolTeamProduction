## 操作方法
### ※コントローラーの接続が必要です
- 移動・メニュー選択

  左スティック
- カメラの回転
  
  右スティック
- ロックオン

  RBボタン・R1ボタン

  Aボタン・×ボタン
- 惑星を投げる ※惑星所有時
  
  Bボタン・〇ボタン  
- 惑星を離す ※惑星所有時

  Yボタン・△ボタン
